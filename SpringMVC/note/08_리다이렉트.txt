# Spring의 리다이렉트

	- 컨트롤러에서 String 타입으로 리턴하면서 redirect: 을 포함하면 리다이렉트를 수행함
	- 리다이렉트시 두 종류의 어트리뷰트를 추가할 수 있음

# RedirectAttribute (또는 Model)

	- redirect 시 클라이언트에서 GET 방식 요청을 하게 되는데 이 요청에 파라미터를 추가하는 방식
	- 주소값 뒤에 어트리뷰트가 추가되는 것을 눈으로 확인할 수 있다
	- 주소값 뒤에 추가되는 방식이기 때문에 자바 객체는 전달할 수 없다
	
# FlashAttribute

	- redirect 시 데이터를 보내고 싶지만 주소에 노출시키고 싶지 않은 경우에 사용하는 어트리뷰트
	- 실제로는 세션을 이용하여 구현되어 있음
		(리다이렉트 응답시 세션에 데이터 저장 -> 리다이렉트 요청 받을시 세션에 있는 데이터 활용 후 삭제)
	- @ModelAttribute(name) 을 통해서 사용하거나 model.getAttribute(name)을 통해 사용할 수 있다
	- 세션에 저장하는 방식이기 때문에 자바 객체 또한 전달할 수 있다
